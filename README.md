# Agrogeo Abstracts Website Content

This repository contains the content for https://agrogeo.curve.space

Content may be built locally and previewed with [MyST markdown](https://mystmd.org). Once you have [installed MyST](https://mystmd.org/guide/installing), simply run `myst start` from within this folder. Once you are happy with the site, any commits to `main` will be automatically pushed to the Agrogeo Abstracts website.
